# SimpleEs
简化相关ES操作

## 半成品，还在完善中
## 目前完成的功能如下：
1. ES连接信息的新增
2. ES连接信息的删除
3. ES连接信息的修改
4. ES索引信息查询
5. ES索引内容查看（完成了查看全部的功能，暂未提供分页）
6. ES连接信息的可视化
7. 自动支持暗黑模式和浅色模式的互换
